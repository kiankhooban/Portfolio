## 🎯 About This Project

This portfolio website is a **fully functional recreation** of the Windows 95 desktop environment, complete with draggable windows, a working Start menu, taskbar, and authentic UI elements. Built entirely from scratch using vanilla JavaScript, HTML, and CSS—no frameworks, just pure nostalgia and code.

## ✨ Features

### 🪟 Authentic Windows 95 Experience
- **Fully Draggable Windows** - Click and drag windows around the desktop just like the real thing
- **Resizable Windows** - Grab any edge or corner to resize windows
- **Minimize/Maximize/Close** - All window controls work perfectly
- **Working Taskbar** - Shows active windows with functional minimize/restore
- **Start Menu** - Classic cascading menu system with hover effects
- **System Clock** - Real-time clock display in the taskbar

### 🎨 Theme Customization
Change the entire color scheme through the Control Panel! Available themes:
- **Original** - Classic Windows 95 teal desktop
- **Rose** - Elegant rose-tinted theme
- **Eggplant** - Deep purple aesthetic
- **Pumpkin** - Warm orange vibes
- **Marine** - Cool ocean blues
- **Raindrop** - Fresh aqua tones

Themes are saved to `localStorage` and persist across sessions.

## 🛠️ Technical Highlights

### Pure Vanilla JavaScript
- **Object-Oriented Design** - Desktop management handled by a comprehensive `DesktopManager` class
- **Event-Driven Architecture** - Efficient event listeners for drag, resize, and window management
- **Z-Index Management** - Automatic window layering and focus handling
- **LocalStorage Integration** - Theme preferences persist across sessions

### CSS Mastery
- **CSS Variables** - Complete theme system powered by custom properties
- **Gradient Effects** - Authentic Windows 95 title bar gradients
- **Inset Borders** - Pixel-perfect recreation of classic UI bevels
- **Responsive Design** - Works on modern screens while maintaining retro aesthetic

### Custom Font
Uses the **W95FA** font for authentic Windows 95 typography

## 🎮 How to Use

1. **Wait for Boot** - Enjoy the nostalgic BIOS sequence
2. **Explore Desktop Icons** - Double-click any icon to open a window
3. **Use Start Menu** - Click the Start button for cascading menus
4. **Drag Windows** - Click and drag title bars to move windows
5. **Resize Windows** - Grab edges or corners to resize
6. **Change Themes** - Open Control Panel to customize colors
7. **Check Taskbar** - Minimize windows and restore them from the taskbar
